# shopify-ready/

Shopify-importable assets generated from the recovered store data. **Start with `SETUP-GUIDE.md`.**

| File | What to do with it |
|---|---|
| `SETUP-GUIDE.md` | Step-by-step: create stores, import, source suppliers, payments, pixels, launch. |
| `store-1-glowlab/products-shopify-import.csv` | Shopify → Products → Import (GlowLab). |
| `store-2-saunazen/products-shopify-import.csv` | Shopify → Products → Import (SaunaZen). |
| `store-3-linguaflow/products-shopify-import.csv` | Shopify → Products → Import (LinguaFlow — launch first). |
| `store-*/content-pack.md` | Product descriptions, SEO fields, and a map of which recovered file goes where. |
| `build-shopify-assets.js` | Regenerate everything: `node shopify-ready/build-shopify-assets.js`. |

Descriptions use compliance-safe language with disclaimers. Add product images (from your supplier app) and verify all claims before going live. Revenue projections in the root `LAUNCH_REPORT.md` are estimates, not guarantees.
