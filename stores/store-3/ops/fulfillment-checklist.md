# LinguaFlow — Order Fulfillment SOP

**Platform:** Shopify + DSers | **Supplier:** CJ Dropshipping | **Updated:** June 2026

---

## DAILY ROUTINE

### Step 1 — Order Review (Morning)
- [ ] Shopify Admin → Orders → Unfulfilled (last 24 hours)
- [ ] Fraud check: flag orders with mismatched billing/shipping, multiple orders same IP, high-value orders from new accounts
- [ ] IMPORTANT: Electronics have higher chargeback rates — apply stricter fraud thresholds

### Step 2 — Push to DSers
- [ ] Select all verified orders → Order to CJ Dropshipping
- [ ] Confirm all show "Ordered" status
- [ ] Electronics note: CJ typically ships earbuds within 24–48 hours, delivery 8–13 days

### Step 3 — Pairing & App Instructions
- [ ] All orders trigger automated email with setup instructions (configure in Klaviyo)
- [ ] Proactively email first-time buyers at Day 1, Day 3 post-delivery with setup tips
- [ ] Earbuds require app pairing — poor initial setup is top support issue; prevent with proactive comms

### Step 4 — Tracking
- [ ] Import tracking numbers as soon as CJ provides them (usually 48–72 hours)
- [ ] Earbuds are higher-value — customers expect faster tracking updates
- [ ] Set up Aftership or Shop app for branded tracking experience

---

## CUSTOMER SUPPORT PRIORITIES FOR ELECTRONICS

**SETUP ISSUES (Most Common)**
Resolution: Send video tutorial link + troubleshooting guide
Common issues: pairing failure, app not recognizing device, audio quality questions
Response time target: 2 hours

**TRANSLATION ACCURACY COMPLAINTS**
Resolution: Confirm they have latest firmware; explain accuracy varies by dialect/noise
Do not promise accuracy in noisy environments
Response time: 4 hours

**DEFECTIVE UNIT**
Resolution: Reship replacement within 24 hours — don't wait for return
Submit CJ dispute immediately with description
Response time: 2 hours — electronics defects require urgency

**BATTERY NOT CHARGING**
Resolution: Check USB-C cable (common culprit), then reship case if issue persists
Submit CJ dispute for manufacturing defect
